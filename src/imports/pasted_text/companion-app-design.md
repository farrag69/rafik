Design a world-class premium production-ready mobile application called "رفيق في الدرب" (Companion on the Journey).

This is NOT a simple Quran or prayer app. It is a long-term daily spiritual companion that helps users build a peaceful, consistent relationship with Quran, prayer, and remembrance of Allah.

------------------------------------------------

CORE PRODUCT VISION

The app must feel:
- Calm
- Premium
- Minimal
- Spiritual
- Warm
- Respectful
- Timeless

Avoid:
- Over decoration
- Islamic clichés
- Gaming mechanics
- Aggressive UI
- Crowded layouts

This should feel like a peaceful digital sanctuary, not a dashboard or social app.

------------------------------------------------

BRAND PERSONALITY

The app behaves like a gentle companion:
- Encourages without judgment
- Never makes user feel guilty
- Uses soft, respectful language
- Focuses on consistency, not pressure

------------------------------------------------

PLATFORMS

- Mobile first (Android + iOS)
- Responsive Web
- Tablet support

Base design width: 360–430dp

Must scale cleanly.

------------------------------------------------

DESIGN SYSTEM

Colors:
Primary: #0F766E (Emerald Green)
Accent: #B68D40 (Gold, very limited use)

Neutral light: warm ivory surfaces
Neutral dark: deep charcoal (not pure black)

Typography:
- UI: Cairo / Noto Sans Arabic
- Quran: dedicated Uthmani font

Style:
- Large rounded cards
- Soft shadows
- Generous whitespace
- Minimal modern icons

------------------------------------------------

COMPONENT SYSTEM (MUST BE CREATED FIRST)

Buttons:
Primary, Secondary, Ghost, Disabled, Loading

Cards:
Hero Cards, Standard Cards, Stats Cards, Feature Cards

Navigation:
Floating Bottom Navigation (5 tabs)

Inputs:
Search, text fields, toggles, selectors

Feedback:
Bottom sheets, dialogs, toasts

Progress:
Circular rings, linear bars, achievements

Special Components:
Quran verse block
Surah list item
Azkar card
Prayer card
Tasbeeh counter
Audio player (mini + full)

All components must support:
- Light/Dark mode
- Empty state
- Loading state
- Disabled state
